## Install
To set up your development environment, run
```bash
  alias config='/usr/bin/git --git-dir=$HOME/.dotfiles/ --work-tree=$HOME'
  curl -sSL https://raw.githubusercontent.com/kensupermen/dotfiles/master/.install_dotfiles.sh | sh
```
